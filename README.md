# 📊 E-Commerce Sales & Profit Dashboard (Excel BI Project)

## 🧠 Project Overview

This project presents an end-to-end Business Intelligence solution built in Microsoft Excel to analyze the sales and profitability of an e-commerce business. The objective was to transform raw transactional data into meaningful insights using data cleaning, modeling, and visualization techniques.

The dashboard provides a comprehensive view of business performance, enabling stakeholders to understand sales trends, customer behavior, and profitability drivers.

---

## 🎯 Objectives

* Analyze sales and profit performance across different dimensions
* Identify key revenue-driving products and categories
* Evaluate regional and customer segment performance
* Understand factors impacting profitability
* Build an interactive dashboard for business decision-making

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* Power Query (Data Cleaning & Transformation)
* Power Pivot (Data Modeling)
* DAX (Data Analysis Expressions)
* Pivot Tables & Charts (Visualization)

---

## 📂 Dataset Description

The dataset used is the Superstore Sales dataset, containing transactional data including:

* Orders (Sales, Profit, Quantity, Discount)
* Customer Information (Segment, Region, Location)
* Product Details (Category, Sub-Category, Product Name)
* Time Data (Order Date, Ship Date)

---

## 🔄 Project Workflow

### 1️⃣ Data Cleaning (Power Query)

* Removed duplicates and handled missing values
* Standardized data formats and column types
* Created calculated columns (Profit Margin, Year, Month)

### 2️⃣ Data Modeling

* Built a **Star Schema**:

  * Fact Table: Orders
  * Dimension Tables: Customers, Products, Date
* Established relationships between tables

### 3️⃣ DAX Measures

Key KPIs created:

* Total Sales
* Total Profit
* Total Orders
* Total Quantity
* Profit Margin
* Average Order Value

### 4️⃣ Dashboard Development

Developed an interactive dashboard including:

* KPI Cards (Sales, Profit, Orders, Margin)
* Sales Trend Over Time (Line Chart)
* Sales by Category
* Profit by Region
* Sales by Customer Segment
* Top Products Analysis
* Interactive slicers (Year, Region, Category, Timeline)

---

## 📊 Key Insights

* Technology category is the primary revenue driver
* Consumer segment contributes the highest sales
* Sales show seasonal trends with peak periods
* A small number of products contribute significantly to revenue
* Profit margins are relatively low despite strong sales

---

## 💡 Business Recommendations

* Optimize discount strategies to improve profitability
* Focus on high-performing product categories
* Expand operations in high-profit regions
* Improve targeting of underperforming customer segments
* Optimize product portfolio based on performance
